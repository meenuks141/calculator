# Calculator

A basic calculator built using HTML, CSS, and JavaScript.

This project was created as part of The Odin Project Foundations curriculum. The calculator performs basic arithmetic operations and handles common edge cases such as consecutive operators and division by zero.

##  Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear button
- Handles consecutive operators correctly
- Prevents division by zero
- Rounds long decimal results
- Responsive layout

##  Built With

- HTML
- CSS
- JavaScript (Vanilla JS)

##  What I Learned

- DOM manipulation
- Event handling
- Managing application state
- Handling edge cases in user input
- Structuring JavaScript logic for UI interaction
